# GithubSearcher
Githubのリポジトリをインクリメンタルサーチで検索するアプリです

## 概要
- SearchViewによってインクリメンタルサーチを実装しました
- キーワードを入力すると5件までリポジトリが表示されます
- 入力確定後はリポジトリ名，リポジトリ作成日，概要が表示されます

## 開発環境
- Android Studio 2.3.3
  - MIN SDK 21
  - MAX SDK 25
- Galaxy S7 Edge(Android 7.0)
